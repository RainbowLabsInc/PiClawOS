## CRITICAL_INVARIANTS
# These MUST be true at all times. Violation = broken build.

INV_001: pyproject.toml build-backend MUST be "setuptools.build_meta"
  WRONG: "setuptools.backends.legacy:build"
  RIGHT: "setuptools.build_meta"
  REASON: Python 3.13 incompatibility causes pip install -e to fail

INV_002: No double nesting in piclaw-src
  WRONG: boot/piclaw/piclaw-src/piclaw/piclaw/ exists
  RIGHT: boot/piclaw/piclaw-src/piclaw/ contains __init__.py directly
  FIX: make sync (uses shutil.copytree, not cp -r)

INV_003: agent.py must NOT use "callable | None" type hint
  WRONG: on_token: callable | None = None,
  RIGHT: on_token=None,  # Optional[Callable]
  REASON: Python 3.13 TypeError

INV_004: api.py MUST use require_auth not verify_token
  WRONG: Depends(verify_token)
  RIGHT: Depends(require_auth)
  REASON: verify_token does not exist in auth.py → NameError on startup

INV_005: api.py MUST have __main__ block
  CODE: |
    if __name__ == "__main__":
        import uvicorn
        from piclaw.config import load as _load
        _cfg = _load()
        uvicorn.run("piclaw.api:app", host=_cfg.api.host or "0.0.0.0",
                    port=_cfg.api.port or 7842, log_level="info", reload=False)
  REASON: Without it, python -m piclaw.api exits immediately

INV_006: config.py CONFIG_DIR must use file-existence check
  RIGHT: |
    def _resolve_config_dir() -> Path:
        if Path("/etc/piclaw/config.toml").exists():
            return Path("/etc/piclaw")
        if Path("/etc/piclaw").exists() and os.getuid() == 0:
            return Path("/etc/piclaw")
        return Path.home() / ".piclaw"
  REASON: Non-root users read ~/.piclaw → doctor shows default values

INV_007: llm/multirouter.py MUST import Path
  ADD: from pathlib import Path
  REASON: NameError when global_cfg.llm.model is a Path string

INV_008: llm/local.py model path MUST end in .gguf and use gemma
  RIGHT: DEFAULT_MODEL_PATH = Path("/etc/piclaw/models/gemma-2b-q4.gguf")

INV_009: memory/qmd.py SEARCH_TIMEOUT MUST be <= 6
  RIGHT: SEARCH_TIMEOUT = 6
  REASON: Agent has 8s timeout on memory.enrich()

INV_010: agent.py cmd_chat MUST call agent.boot() before agent.run()
  CODE: agent = Agent(cfg); await agent.boot()
  REASON: LLM router not initialized → "No LLM backends configured"

INV_011: llm/registry.py bootstrap MUST allow local/ollama without API key
  WRONG: if not llm.api_key: return False
  RIGHT: if not llm.api_key and llm.backend not in ("local", "ollama"): return False

INV_012: llm/multirouter.py health_check MUST check file existence for local
  CODE: |
    async def health_check(self) -> bool:
        if self.global_cfg and self.global_cfg.llm.backend == "local":
            return self._local.model_path.exists()
        return bool(self.registry.list_enabled()) or self._local_loaded

INV_013: wizard.py MUST flush stdin before every input()
  CODE: |
    def _flush_stdin():
        try: import termios; termios.tcflush(sys.stdin, termios.TCIFLUSH)
        except Exception: pass
  REASON: SSH buffering causes input fields to be skipped

INV_014: cli.py chat MUST connect via WebSocket API first, fallback to direct
  LOGIC: if api_running(cfg) → ws://127.0.0.1:{port}/ws/chat?token={token}
         else → Agent(cfg) + await agent.boot() [shows "(Offline-Modus)"]

INV_015: QMD MUST NOT run on every chat turn
  WRONG: await self.qmd.update() after each extract_and_index()
  RIGHT: Hourly systemd timer with nice 19
  REASON: QMD spawns Node.js process using 200%+ CPU for 2-4 minutes

INV_016: SOUL.md MUST be excluded from QMD memory index
  WHERE: memory/qmd.py setup_collections() and _grep_fallback()
  REASON: Soul injected as "memory" context confuses model on unrelated queries
  STATUS: ✅ Implemented - SOUL_FILENAME = "SOUL.md" excluded

INV_017: Marketplace query MUST be cleaned before search
  WHERE: tools/marketplace.py _clean_query() and marketplace_search()
  KEY: PLZ extracted from query string if not passed as location param
  KEY: Noise phrases sorted by length DESC (longer phrases removed first)
  KEY: Use lookahead/lookbehind for PLZ regex, not \b

INV_018: agent.py marketplace shortcut MUST call handler directly when possible
  LOGIC: |
    if mp_kwargs.get("query") and (mp_kwargs.get("location") or mp_kwargs.get("radius_km")):
        handler = self._handlers.get("marketplace_search")
        return await handler(**mp_kwargs, notify_all=True)  # direct, fast
    return await self._delegate_to_search_assistant(user_input)  # sub-agent
  REASON: Sub-agent delegation was unreliable for simple PLZ+query searches

INV_019: install.sh MUST set /etc/sudoers.d/piclaw for service restart
  CONTENT: |
    piclaw ALL=(ALL) NOPASSWD: \
      /bin/systemctl restart piclaw-api, \
      /bin/systemctl restart piclaw-agent, ...
  REASON: piclaw user must restart its own services after 'piclaw update'

INV_020: updater.py MUST use git pull, NOT pip install --upgrade
  FLOW: git pull → pip install -e . (only if pyproject.toml changed) → sudo systemctl restart
  REASON: Pi is an editable install from git repo, not a PyPI package
